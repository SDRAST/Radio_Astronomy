get_3C_coords¶

Radio_Astronomy.vla_cal.get_3C_coords(name)¶

Formatted J2000 right ascension and declination and IAU name

Returns the formatted J2000 right ascension and declination and IAU name given the 3C name. Example

>>> ra,dec,iau = get_3C_coords('3C286')
>>> print ra,dec,iau
13h31m08.287984s 30d30'32.958850" 1331+305

:param name : 3C name, like 3C123

Returns

ra, dec, IAU_name
