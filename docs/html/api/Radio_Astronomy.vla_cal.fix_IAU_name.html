fix_IAU_name¶

Radio_Astronomy.vla_cal.fix_IAU_name(name)¶

Handles cases where the IAU designator is too long or too short.

Returns a name suitable for an index into the dictionaries used in this module.

The calling routine needs to ensure that name at least looks like an IAU designation.
