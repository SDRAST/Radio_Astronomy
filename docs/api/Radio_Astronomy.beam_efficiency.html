beam_efficiency¶

Radio_Astronomy.beam_efficiency(antenna_solid_angle, beam_solid_angle)¶

Fraction of flux in the main beam that registers at the receiver.

Kraus, Radio Astronomy, eq. 6-23.

This is generally larger than the aperture efficiency by a factor of 1.2 - 1.4.
