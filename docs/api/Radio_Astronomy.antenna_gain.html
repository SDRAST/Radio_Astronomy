antenna_gain¶

Radio_Astronomy.antenna_gain(aperture_efficiency, geometrical_area)¶

Antenna gain in K/Jy.

Given the geometrical aperture area in m^2 and the aperture efficiency, this returns the gain in K/Jy

:param aperture_efficiency : :type aperture_efficiency : float

:param geometrical_area : typically pi*(diam/2)^2, same units as wavelength :type geometrical_area : float

Returns

float
