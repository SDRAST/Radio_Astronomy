match_IAU_name¶

Radio_Astronomy.vla_cal.match_IAU_name(name, name_list)¶

Try to match a short IAU name to one in a catalogue

This is done by rounding the catalogue name as much as necessary. For example, if the name is 1012+53, it is one character short of the coorrect name 1012+531. If the name is 1012+5307, it is one character over.
