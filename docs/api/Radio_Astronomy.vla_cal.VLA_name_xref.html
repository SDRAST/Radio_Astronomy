VLA_name_xref¶

Radio_Astronomy.vla_cal.VLA_name_xref(cal_data)¶

Make cross-reference lookup for IAU B names and 3C catalog numbers.

Two dictionaries are returned. The first has IAU B epoch names as keys and the second has 3C names as keys. Both return the corresponding J name.

:param cal_data : VLA calibrator data dictionary :type cal_data : dictionary of dictionaries

Returns

tuple of dictionaries
